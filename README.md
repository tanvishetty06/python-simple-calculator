Simple Calculator (Python)

📌 Project Description

This is a simple command-line calculator built using Python.
It allows the user to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

🚀 Features

- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers
- Handles division by zero errors

🛠 Technologies Used

- Python

▶️ How to Run the Program

1. Install Python on your system.
2. Download the file "simple_calculator.py".
3. Open a terminal or Python IDLE.
4. Run the program.

Example command:

python simple_calculator.py

💡 Example Output

Simple Calculator
Enter first number: 5
Enter second number: 3
Choose operation (+, -, *, /): +
Result: 8
